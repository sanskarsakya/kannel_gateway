## Installatioon
Git clone : repo
### Frontend
1. cd frontend
2. npm install
3. ng serve (localhost:4200)

### backend
1. cd smsfinal
2. mvn clean install
3. mvn spring-boot:run (make sure mysql is active)

### MySQL configuration
1. cd src/main/resources/applicaiton.properties
2. change as per requirement.
